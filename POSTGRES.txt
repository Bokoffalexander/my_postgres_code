ДЛЯ UBUNTU

First, execute the following command to create the file repository configuration:
$ sudo sh -c 'echo "deb http://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list'

Second, import the repository signing key:
$ wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -

Third, update the package list:
$ sudo apt update

Finally, install the latest version of PostgreSQL:
$ sudo apt install postgresql





$ sudo -i -u postgres

To create a new database, in this example named
 mydb, you use the following command:
$ createdb mydb

you can destroy it using the following command:
$ dropdb mydb

Once you have created a database, you can 
access it by:
• Running the PostgreSQL interactive terminal
 program, called psql, which allows you to 
interactively enter, edit,
 and execute SQL commands.
$ psql mydb


psql program has a number of internal
 commands that are not SQL commands. 
They begin with
the backslash character, “\”


psql

SELECT version();

SELECT current_date;

SELECT 2 + 2;

you can get help on the syntax of various PostgreSQL SQL
commands by typing:
\h

\l   //это список баз данных

\connect testdb    //это подключиться к базе с именем db_name

\du  //– список users
\db //– список БД
\dt      //– список таблиц

To get out of psql, type:
\q


Создание и настройка базы данных
запустить консоль psql: sudo -u postgres psql
создать базу данных: CREATE DATABASE <db_name>;
создать пользователя: CREATE USER <username> WITH PASSWORD '<password>';
предоставить пользователю административный доступ к базе данных:
 GRANT ALL PRIVILEGES ON DATABASE <db_name> TO <username>;
